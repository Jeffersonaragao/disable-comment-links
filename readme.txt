== Description ==

The Disable Comment Links plugin prevents users from adding hyperlinks within comments on your WordPress site. By filtering comment content, the plugin detects any presence of links and prohibits their submission, helping you combat spam and enhance security.

== Installation ==

1. Upload the `disable-comment-links` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Can users still post comments? =
Yes, users can post comments without including any hyperlinks. The plugin only restricts comments that contain links.

= How does the plugin handle existing comments with links? =
The plugin doesn't affect existing comments. It focuses on preventing new comments with hyperlinks from being submitted.
